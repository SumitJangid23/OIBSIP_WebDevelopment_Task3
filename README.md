Tmperature Converter

Objective

The objective of this task is to develop a temperature converter that converts temperature values between Celsius, Fahrenheit, and Kelvin.

Steps Performed

- Created the basic structure using HTML
- Designed the interface using CSS
- Implemented temperature conversion logic using JavaScript
- Added validation for user input

Tools Used

- HTML
- CSS
- JavaScript
- Visual Studio Code

Outcome

Successfully developed a temperature converter that accurately converts temperature values between different units.
